FolderPeek 0.3 · macOS Quick Look Extension

Preview folders before opening them.

FolderPeek shows folder, zip, and tar contents directly from Finder Quick Look, so you can find the right material without opening every folder first.

Download for macOS View on GitHub

Latest release: 0.3

What it previews

Folders

File names, sizes, modified dates, type grouping, and thumbnails when available.

Archives

Flat zip and tar listings without extracting the archive to disk.

Finder workflow

Select a folder or archive, press Space, and inspect the contents before opening it.

Install

FolderPeek currently ships as an early-tester direct download. Developer ID signing and notarization are not complete yet, so macOS Gatekeeper may require manual approval on first launch.

Download FolderPeek-0.3.zip from GitHub Releases.
Unzip it.
Move FolderPeek.app to /Applications.
Launch FolderPeek once.
In Finder, select a folder, .zip, or .tar file and press Space.

Homebrew

Power users can install through the personal Homebrew tap.

brew tap doma17/folderpeek
brew trust doma17/folderpeek
brew install --cask folderpeek
